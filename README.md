# 🎓 EduTrack – Student Management System

## 📌 Introduction

EduTrack is a full-stack student management system developed using Spring Boot and MySQL.
It helps manage users, authentication, and future modules like student data, attendance, and marks.

---

## 🚀 Features

* User Registration
* User Login
* REST API Integration
* Frontend UI (HTML, CSS, JavaScript)
* MySQL Database Integration

---

## 🛠 Tech Stack

* Backend: Spring Boot (Java)
* Database: MySQL
* Frontend: HTML, CSS, JavaScript
* Build Tool: Maven

---

## 📂 Project Structure

```
edutrack/
 ├── src/                # Backend (Spring Boot)
 ├── frontend/           # Frontend UI
 ├── database.sql        # Database schema
 ├── pom.xml
 ├── README.md
```

---

## 🔗 API Endpoints

* POST /auth/register → Register user
* POST /auth/login → Login user
* GET /auth/users → View all users

---

## 🗄 Database

Database Name: **edutrack**
Table: **users**

## ▶️ How to Run

### 1. Run Backend

### 2. Open Frontend
